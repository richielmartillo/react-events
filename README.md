# React Events

Projeto simples desenvolvido com React e Vite para praticar eventos em componentes React.

O objetivo do exercício foi entender como trabalhar com eventos como clique de botão e envio de formulário, além de praticar o uso de componentes reutilizáveis e props.

## Tecnologias utilizadas

- React
- JavaScript
- Vite
- HTML
- CSS
- Git & GitHub

## Funcionalidades

- Exibe botões personalizados na tela
- Usa props para alterar o texto dos botões
- Usa props para alterar a mensagem exibida no alerta
- Executa uma função ao clicar em um botão
- Exibe alertas com mensagens diferentes
- Trabalha com evento de formulário usando `onSubmit`
- Impede o recarregamento da página com `preventDefault()`

## Conceitos praticados

- Criação de componentes React
- Importação e exportação de componentes
- Props
- Eventos em React
- `onClick`
- `onSubmit`
- Funções handler
- Formulários em React
- `preventDefault()`
- Estrutura básica de um projeto com Vite
